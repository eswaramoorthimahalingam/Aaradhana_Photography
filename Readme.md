# Aaradhana Photography

Aaradhana Photography website for wedding cards, printing services, office stationery, and event photography.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the project locally:

```bash
npm start
```

Build for production:

```bash
npm run build
```

## Services

- Wedding cards
- Wedding printing
- General printing
- Office stationery
- Photography
- Contact and price details
